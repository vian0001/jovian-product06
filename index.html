Kesimpulan: Winbox memiliki spesifikasi sistem yang minimal karena dirancang sebagai utility ringan, dapat berjalan di Windows (native), Linux dan macOS (melalui Wine), memerlukan konfigurasi firewall yang tepat, dan untuk versi terbaru Winbox 4 sudah mendukung native pada ketiga platform utama tanpa emulasi.

6. Persiapan Perangkat MikroTik untuk Akses

Sebaiknya login mikrotik melalui webfig atau winbox dilakukan dengan reset konfigurasi default dan tambah pengguna baru serta ubah password admin untuk menjaga keamanan.

Dari sumber MikroTik Documentation (2024), untuk terhubung ke router masukkan IP atau MAC address router, spesifikasikan username dan password (jika ada) dan klik tombol Connect, dapat juga memasukkan nomor port setelah alamat IP dengan memisahkannya dengan titik dua seperti 192.168.88.1:9999. Direkomendasikan menggunakan alamat IP kapan pun memungkinkan karena sesi MAC menggunakan network broadcast dan tidak 100% dapat diandalkan.

Dari sumber Google Play Store (2024), biasanya tidak ada password default (biarkan kosong), atau ditentukan dalam dokumen yang disertakan dengan persyaratan router MikroTik yang menjalankan RouterOS v6 atau yang lebih baru. Default username pada router baru adalah admin dan biasanya tidak ada password default atau disebutkan dalam dokumen yang menyertai perangkat.

Dari sumber MikroTik Documentation (2024), sebagian besar driver jaringan tidak akan mengaktifkan IP stack kecuali perangkat host memiliki konfigurasi IP sehingga perlu mengatur konfigurasi IPv4 pada perangkat host. Kesimpulan: Persiapan perangkat MikroTik untuk akses Winbox meliputi koneksi fisik antara komputer dan router, konfigurasi IP pada komputer host, penggunaan kredensial default (username: admin, password: kosong), dan pemahaman bahwa RouterOS hadir dengan konfigurasi default yang dapat disesuaikan sesuai kebutuhan jaringan.

Dari sumber Network Data Sistem (2021), MAC Address bekerja di layer 2 sehingga dapat mengakses RouterOS tanpa perlu mengonfigurasi IP Address, namun MAC address tidak cukup stabil untuk digunakan secara konsisten dan sebaiknya hanya digunakan di awal konfigurasi. MAC Address sebaiknya hanya digunakan pada awal konfigurasi dan setelah itu disarankan beralih ke IP Address untuk koneksi yang lebih stabil.

Dari sumber ID-Networkers (2022), fitur dasar ketika ingin konek ke router adalah Connect To untuk memasukkan IP Router atau MAC Address Router yang diinginkan dengan Login berupa username dan password login ke router (default = admin) dan Password untuk router yang ingin dikoneksikan (default tidak diisi). Add/Set digunakan untuk menyimpan informasi berupa IP/MAC Router, Login, Password, ke menu Managed, jadi ketika konek ke router kedepannya, tidak perlu ketik lagi untuk efisiensi waktu.

Perlu diingat bahwa sebagian besar driver jaringan tidak akan mengaktifkan IP stack kecuali perangkat host memiliki konfigurasi IP, sehingga perlu mengatur konfigurasi IPv4 pada perangkat host terlebih dahulu.
Kesimpulan: Winbox mendukung dua metode koneksi utama yaitu melalui MAC Address yang bekerja di layer 2 dan cocok untuk konfigurasi awal, serta melalui IP Address yang lebih stabil dan direkomendasikan untuk penggunaan konsisten, dengan persyaratan teknis tertentu seperti MTU 1500 untuk koneksi MAC Address.

8. Login Awal Menggunakan Winbox

Dari sumber Network Data Sistem (2021), jalankan Winbox lalu klik Neighbors agar Winbox dapat menemukan alamat IP dan MAC address dari MikroTik RouterOS yang terhubung ke jaringan melalui broadcast, lalu masukkan dan klik connect kemudian masukkan user login dan password untuk mengakses router, kemudian Winbox akan mengunduh plugin dari router. Neighbors merupakan tombol yang digunakan untuk menemukan dan menampilkan perangkat MNDP (Mikrotik Neighborhood Discovery Protocol) atau CDP (Cisco Discovery Protocol).

Dari sumber Nevacloud (2023), fitur Neighbors digunakan untuk menemukan perangkat MikroTik Neighbor Discovery Protocol (MNDP) atau Cisco Discovery Protocol (CDP) dimana dengan fitur ini menemukan perangkat MikroTik RouterOS yang terhubung ke jaringan sangatlah mudah. Neighbor Discovery adalah salah satu fitur pada Winbox Mikrotik yang berfungsi untuk menemukan perangkat Mikrotik apa saja yang terhubung dengan PC/laptop di jaringan secara otomatis.

Kesimpulan: Pengaturan keamanan awal pada Winbox meliputi mengubah password default admin dengan password yang kuat, mengaktifkan protokol keamanan seperti SSH, membuat manajemen pengguna dengan hak akses terbatas sesuai kebutuhan, mengaktifkan audit logging, dan selalu menggunakan versi Winbox terbaru yang dilengkapi dengan enkripsi AES128-CBC-SHA dan sertifikat keamanan resmi.
